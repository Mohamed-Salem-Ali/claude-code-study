# Experiments

Free sandbox — try anything here without worrying about breaking the main project.

---

## Guidelines

- No rules. Break things. That's the point.
- If something works well, move it to `templates/` or document it in `notes/`.
- Folders are disposable. Create and delete freely.

---

## Ideas to Experiment With

- Try giving Claude a terrible CLAUDE.md and compare quality vs. a good one
- Test how far you can push custom commands (multi-step, conditional logic)
- Try the SDK to run automated tasks on uigen
- Explore what happens when context fills up (don't use /compact — let it overflow)
- Build a custom MCP server for something you use daily
